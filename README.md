
# Offline AI Tutor (GitHub Pages)

## Setup for Teachers / Admin
1. Push this folder to a GitHub repository.
2. Go to **Settings → Pages**.
3. Select branch `main` and folder `/ (root)`.
4. GitHub provides a URL (e.g., `https://yourusername.github.io/offline-ai-tutor/`).
5. Share this URL with students.

## Instructions for Students
1. Open the URL in Safari (iPhone/iPad).
2. Tap **Share → Add to Home Screen**.
3. Launch the app.
4. First launch downloads AI model (~200-400MB) – take a few minutes.
5. After that, the app works fully offline.
6. Type your questions in the input box and get step-by-step hints.
